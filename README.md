# java-china

## 特性

- 界面简洁清爽，对移动端友好
- 支持markdown语法
- 支持Emoji表情输入
- 支持Github账户登录
- 支持@用户
- 支持在线播放音乐(小彩蛋)
- 每日励志名言
- 更多功能还在开发...
